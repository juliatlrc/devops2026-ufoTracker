# UFO Tracker – Operação Ufology

## Sobre o projeto
Aplicação Spring Boot para rastreamento de avistamentos de OVNIs,
com PostgreSQL, Redis e deploy em Kubernetes.

## Papel do Git no DevOps
O Git é fundamental no ciclo DevOps pois permite rastrear todas as
mudanças no código, reverter erros e colaborar em equipe. Cada commit
representa um estado estável do projeto, viabilizando automação de
entregas via CI/CD.

## Branches e Tags
- **main**: branch principal com código estável
- **ci/setup**: branch de configuração dos pipelines de CI/CD
- **Tags**: marcam versões estáveis (ex: v1.0.0) usadas em releases

## CI/CD com GitHub Actions
Os workflows automatizam build, testes e deploy a cada push ou PR.
